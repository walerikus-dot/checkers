01 · Игровая логика
🤖 Claude Code пишет код
Ядро игры
  • Движок русских шашек (8×8) — все правила, ходы, взятия, дамкиполностью генерируется
  • Международные шашки (10×10) — адаптация правилна основе ядра
  • Английские шашки — отдельная конфигурация правил
  • Валидация ходов на сервере — NestJS сервис
  • Обнаружение конца партии — нет ходов, нет шашек, ничья
  • Запись истории ходов — структура данных, сохранение
Настройки партии
  • Конфигурируемые правила — JSON-схема параметров лобби
  • Логика таймера — блиц, контроль времени, просрочка

02 · AI-противник
🤖 Claude Code пишет код
Все уровни AI
  • Лёгкий — случайные ходы с фильтрацией допустимых
  • Средний — полный Minimax с эвристической оценкой позиции

  • Opening book — таблица дебютов в JSON

🤖 Claude Code пишет код
NestJS приложение

  • Glicko-2 алгоритм — полная реализация расчёта рейтинга
  • Matchmaking логика — очереди, фильтрация, тайм-аут
  • Турнирная сетка — алгоритмы Single/Double Elimination, Round Robin
  • Rate limiting — middleware конфигурация
